# sector_tracker.py 사용 안내

반도체 트래커(semicon_tracker.py)와 동일한 구조로, 국내 상장 8개 섹터의
수출 트렌드를 관세청 TRASS API로 자동 수집한다.

## 대상 섹터
K뷰티화장품 · 이차전지 · 자동차 · 철강 · 석유화학 · 조선 · 의료기기 · K푸드

각 섹터마다 두 시트가 생성된다.
- `{섹터}_제품` : 서브카테고리(품목)별 월별 수출액·중량·ASP·MoM·YoY
- `{섹터}_국가` : 대표 HS 향지(국가)별 월별 수출액·비중·MoM·YoY
- 맨 앞 `Cross_Sector_Dashboard` : 8섹터 최신월 수출과 YoY/MoM 한눈에 비교

## 실행

1. 파일 상단 `SERVICE_KEY` 에 발급키 입력 (반도체 트래커와 동일 키 사용 가능)
2. 최초 1회
   ```python
   MODE   = "full"
   MONTHS = 90
   ```
   Run. `data_sector/` 폴더에 섹터별 캐시 CSV 저장 후 엑셀 생성.
3. 매월 업데이트
   ```python
   MODE = "update"
   ```
   Run. 최근 2개월만 호출해 캐시에 누적.

## 호출량 / 소요시간
- full(90개월) 전체 약 5,400콜 → 약 30~45분 (개발계정 일 1만콜 한도 내)
- update(2개월) 약 120콜 → 1분 내
- 너무 오래 걸리면 `SECTORS_TO_RUN` 에서 섹터를 줄여 나눠 실행 가능

## HS 코드 주의
대부분 6자리 코드는 정확하나, 일부 소재 코드는 근사치다 (config 에 `(*)` 표시).
특히 이차전지 양극재/분리막은 분류가 다양하므로 관세청 'HS CODE 내비게이션'
(tradedata.go.kr > 사용자지원)에서 검증 후 `SECTORS` 딕셔너리를 조정 권장.

## 섹터/품목 추가·수정
`SECTORS` 딕셔너리만 편집하면 시트가 자동으로 따라 생성된다.
```python
"신규섹터": {
    "groups": {"품목라벨": ["HS코드", ...], ...},
    "country_hs": ["향지분해용 대표코드"],
    "key_countries": ["US", "CN", ...],
    "companies": "대표 상장사",
}
```

## 활용 팁
- 관세청 10일·20일 잠정치(매월 11/21/익월1일)로 분기 실적 방향을 가장 빠르게 포착
- 제품믹스 시트의 ASP(USD/kg) 추세는 단가 회복/하락 신호로 활용
- 국가 시트의 비중(%) 변화는 채널 다각화(예: K뷰티 미국 비중 급등)를 선행 포착
- Cross_Sector_Dashboard 의 YoY 를 KOSPI 섹터 수익률과 나란히 놓고 수출 선행성 검증
