# 섹터 수출 트래커 (sector_tracker)

국내 상장 8개 섹터의 수출 트렌드를 관세청 TRASS API로 자동 수집·집계한다.
반도체(`semi/`)는 더 깊게 보는 별도 심층 모델로 분리되어 있다 — 이 폴더는
8개 섹터를 넓게 훑는 **breadth 모니터**.

## 폴더 구조
```
export_data_all/
├─ sector_tracker.py     # 엔진: 수집 · 파생지표 · 엑셀 빌드
├─ sectors_config.py     # 섹터 정의(HS 코드 매핑) — 리서치 자산, 여기만 편집
├─ data_sector/          # 섹터별 캐시 CSV (원천 데이터, 건드리지 말 것)
├─ output/               # 결과 엑셀 (sector_tracker.xlsx)
├─ _archive/             # 수동 백업 등 보관
└─ semi/                 # 반도체 = 별도 심층 모델 (자체 코드·캐시·출력)
```

## 대상 섹터 (15개)
**기존 8:** K뷰티화장품 · 이차전지 · 자동차 · 철강 · 석유화학 · 조선 · 의료기기 · K푸드
**확장 7:** 방산 · 전력기기·변압기 · 제약·바이오 · 엔터·K콘텐츠 · 태양광·신재생 · 기계·건설기계 · 타이어

> 일부 신규 섹터는 HS 매핑·관세청 반영에 한계가 있다 (`sectors_config.py` 주석 참조):
> 방산=G2G·기밀로 월별 들쭉날쭉, 제약·바이오/엔터=기술수출·방영권은 미반영(실물 수출만),
> 태양광=HS2022 개편으로 2022년 이후만 깔끔. HS코드 `(*)`는 검증 후 조정 권장.

각 섹터마다 두 시트 + 맨 앞 크로스섹터 대시보드가 생성된다.
- `{섹터}_제품` : 서브카테고리(품목)별 월별 수출액·중량·ASP·MoM·YoY
- `{섹터}_국가` : 대표 HS 향지(국가)별 월별 수출액·비중·MoM·YoY
- `Cross_Sector_Dashboard` : 8섹터 최신월 수출과 YoY/MoM 한눈에 비교

## 실행 (VS Code 에서 파일 상단 상수만 수정 후 Run)

| MODE | 동작 | API 호출 |
|------|------|----------|
| `"full"`   | 최초 전체 수집 (MONTHS 개월) → 캐시 저장 → 엑셀 | 많음 (~5,400콜/30~45분) |
| `"update"` | 매월 최근 2개월만 수집해 캐시에 누적 → 엑셀     | 적음 (~120콜/1분) |
| `"cache"`  | **API 없이** `data_sector/` 캐시만 읽어 마스터 엑셀 | 없음 (즉시) |

- `SERVICE_KEY` 에 발급키 입력 (반도체 트래커와 동일 키 사용 가능). `cache` 모드는 키 불필요.
- `SECTORS_TO_RUN` 에서 수집할 섹터를 고른다 (full/update 전용). **`cache` 모드는 이 값과 무관하게 캐시에 있는 모든 섹터를 합친다.**

### 전형적 워크플로
1. 최초: `MODE="full"`, `SECTORS_TO_RUN`=전체 → 한 번 돌려 캐시 구축.
2. 매월: `MODE="update"` → 신규 월만 캐시에 누적.
3. 리포트만 다시: `MODE="cache"` → API 안 때리고 8섹터 마스터 엑셀 재생성.

CLI 로도 가능: `python sector_tracker.py --mode cache`

## HS 코드 주의
대부분 6자리 코드는 정확하나 일부 소재 코드는 근사치다 (`sectors_config.py` 에 `(*)` 표시).
특히 이차전지 양극재/분리막은 분류가 다양하므로 관세청 'HS CODE 내비게이션'
(tradedata.go.kr > 사용자지원)에서 검증 후 `sectors_config.py` 의 `SECTORS` 조정 권장.

## 섹터/품목 추가·수정
`sectors_config.py` 의 `SECTORS` 딕셔너리만 편집하면 시트가 자동으로 따라 생성된다.
```python
"신규섹터": {
    "groups": {"품목라벨": ["HS코드", ...], ...},
    "country_hs": ["향지분해용 대표코드"],
    "key_countries": ["US", "CN", ...],
    "companies": "대표 상장사",
}
```

## 활용 팁
- 관세청 10일·20일 잠정치(매월 11/21/익월1일)로 분기 실적 방향을 가장 빠르게 포착
- 제품믹스 시트의 ASP(USD/kg) 추세는 단가 회복/하락 신호로 활용
- 국가 시트의 비중(%) 변화는 채널 다각화(예: K뷰티 미국 비중 급등)를 선행 포착
- Cross_Sector_Dashboard 의 YoY 를 KOSPI 섹터 수익률과 나란히 놓고 수출 선행성 검증
